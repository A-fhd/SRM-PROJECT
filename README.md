Student Record Management System (SRMS)

This project is a simple Java console application used to manage student records.
It allows adding students, viewing records, and saving data to a file.

Requirements:
1-NetBeans IDE
2- VS code
3-Cursor

How to Run:
1. Open the project in NetBeans or VS code  or Cursor
2. Run the Main class.
3. Follow the menu instructions.

Features:
- Object-Oriented Programming
- Exception Handling
